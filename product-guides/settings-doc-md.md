# Settings Doc md

## Navigating and configuring settings in LineCraft.ai

LineCraft.ai provides a centralized **Settings** area where you can manage your user profile, operational preferences, shift schedules, production targets, fault code mappings, loss types, and process parameters. Whether you're updating personal information or configuring plant-level data, all of these options are accessible from the same secondary navigation bar. This guide walks you through each section step by step, so you can tailor the platform to your operational needs.

***

### Prerequisites

* You must have an active LineCraft.ai account with sufficient permissions to edit settings.
* Access the **Settings** area from the top menu before following any steps in this guide.

***

### Managing your user profile

#### Viewing and editing personal information

Navigate to **Settings > Profile** to view your personal information, including your email address, name, and phone number.

To make changes, click the **Edit** button next to Personal Information. The **Edit Personal Information** dialog opens, where you can update the **Name** field (for example, changing it from "Mary Jane" to "Mahima Gautam").

![Profile section showing personal information and the Edit button](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_1JYsfQg9PZGeYY5ZG/abccc9a4454ccf2f3c0bc6063914d694profile_section_showing_personal_information_and_the_edit_button.jpg)

![Edit Personal Information dialog with name and phone number fields](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_ZbmAKPXhI4uEAkJZj/66679b2440d9667bb0c622c5bdb7fb38edit_personal_information_dialog_with_name_and_phone_number_fields.jpg)

#### Updating your phone number

In the **Edit Personal Information** dialog, click the **Select Code** dropdown under Phone Number to choose your country code (for example, +91 for India or +1 for USA), then enter your new phone number.

![Country code dropdown expanded for phone number selection](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_sQlag9UCUcxL9qiy2/6abfaa0b6d42788e84e66eaa4bf0fb7fcountry_code_dropdown_expanded_for_phone_number_selection.jpg)

#### Changing your password

Click the **Change Password** button in the Profile section. In the **Change Password** dialog, enter your **Current Password**, a **New Password**, and then re-enter the new password to confirm.

Your new password must meet the following requirements:

* At least 8 characters
* At least 1 uppercase letter
* At least 1 lowercase letter
* At least 1 number

Click **Save** to apply the change, or **Cancel** to exit without saving.

![Change Password dialog with password requirements and input fields](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_V75tkm4KZTYQ49grQ/e94aca66f069d8092be58b7b33e3acf5change_password_dialog_with_password_requirements_and_input_fields.jpg)

### Setting preferences

Navigate to **Settings > Preferences** to configure your display options. Under **Time Format**, click the dropdown to select your preferred format from the available options: Default, HH:MM:SS, or Seconds.

![Preferences section with Time Format dropdown expanded](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_hYW0KYfIh85Jc5nvf/9742564f60a2e2bbdc07c3fea5605c05preferences_section_with_time_format_dropdown_expanded.jpg)

### Managing shifts

#### Viewing shift groups

Navigate to **Settings > Shifts** to see a list of all shift groups, along with their default status, actions, shifts, and breaks. You can set any shift group as the default by clicking the toggle in the **Default** column.

![Shifts section showing shift groups, shifts, and breaks](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_RJ4gvyDkiZ3Lst0wk/3a56b62f3f608cd44d2b3c948b9f593bshifts_section_showing_shift_groups__shifts__and_breaks.jpg)

#### Setting a default shift group

When you click the **Default** toggle for a shift group, the **Set as Default Shift Group** dialog appears, showing the shift and break details for that group. Review the information, then click **Set as default** to confirm, or cancel to exit.

![Set as Default Shift Group dialog with shift and break details](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_61nUgJdimVXfRSbnZ/e985868a3b729ed2ac9bc8c935779d73set_as_default_shift_group_dialog_with_shift_and_break_details.jpg)

#### Creating or editing a shift group

Click **Create Shift Group** to add a new group, or select an existing group to edit it. On the Create/Edit Shift Group screen, fill in the following:

* **Shift Group Name** — for example, "current set"
* Optionally, mark the group as the default shift group
* For each shift:
  * **Shift Name** — for example, "shift 1"
  * **Start Time** and **End Time** — for example, 06:30 to 13:45
* For each break:
  * **Break Name** — for example, "Tea Break"
  * **Start Time** and **End Time**

Use the **Add Break** button to include additional breaks, or **Clear All** to remove all breaks at once. Click **Edit Shift Group** or **Create Shift Group** to save your changes.

![Edit Shift Group screen with shift and break configuration](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_2zL1ziifEQ3TJcT7w/f8705b68500f9a883ab49243b7f65490edit_shift_group_screen_with_shift_and_break_configuration.jpg)

### Configuring targets

#### Viewing target JPH values

Navigate to **Settings > Targets** to see a table listing each part type alongside its **Target JPH** (Jobs Per Hour) value. Click the **Edit** button to begin making changes.

![Targets section listing part types and target JPH values](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_Nwe7oqg4yK0FMRxqC/3ba3aee2876599a7313078cbf655d096targets_section_listing_part_types_and_target_jph_values.jpg)

#### Editing target JPH values

On the **Edit Target JPH** screen, update the JPH value for each part type as needed — for example, entering 65. Click **Save Changes** to apply your updates, or **Cancel** to discard them and return to the Targets overview.

![Edit Target JPH screen with editable JPH values for each part type](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_34tYp78he8IgvRrzQ/96d7c81495d4a034b84fdaa7dd51a0efedit_target_jph_screen_with_editable_jph_values_for_each_part_type.jpg)

### Mapping fault codes

The **Fault Code Mapping** section lets you associate fault codes with specific assets or zones. Navigate to **Settings > Fault Code Mapping** to access it. The screen contains two sub-tabs: **Asset Mapping** and **Zone Mapping**.

#### Understanding mapping status

When you open the **Asset Mapping** sub-tab, a status bar shows how many assets have been mapped — for example, "Fault code mappings added for all assets: 277/277 mapped." Switching to the **Zone Mapping** sub-tab shows the equivalent status for zones — for example, "Fault code mappings pending for some zones: 1/46 remaining."

![Fault Code Mapping screen with Asset Mapping tab active, showing mapping status and asset list](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_XE3WX164awZuN2uWF/a5bb0b88fb70fd03b31dac7e48c68692fault_code_mapping_screen_with_asset_mapping_tab_active__showing_mapping_status.jpg)

![Zone Mapping tab active, showing pending mappings and zone list](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_QSJAVMDEUCX4PVjAB/22e80b42749a5106267579af62bea1bfzone_mapping_tab_active__showing_pending_mappings_and_zone_list.jpg)

#### Uploading a fault code mapping for a single asset

In the **Asset Mapping** sub-tab, each listed asset (for example, 0009\_01TR\_ST01) has two options: **Download Existing Mapping** and **Upload New Mappings**. Click **Upload New Mappings** next to the asset you want to configure.

The **Upload Fault Code Mapping** dialog appears, showing the asset name and the following upload requirements:

1. Every row in the sheet must be unique to map configurations correctly.
2. All columns in the sheet are mandatory.
3. No empty fields are allowed in the sheet.

Click **Upload Fault Code Sheet** to proceed, or **Cancel** (or the **X** button) to close the dialog without uploading.

![Upload Fault Code Mapping dialog with asset name and upload instructions](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_LiTDoeCsuYU6niE31/1dda9da46f65a3ea72d9c5fa49bf9d7bupload_fault_code_mapping_dialog_with_asset_name_and_upload_instructions.jpg)

#### Uploading fault code mappings for multiple assets at once

In the **Asset Mapping** sub-tab, click the **Bulk Upload for Multiple Assets** button on the right side of the screen. The bulk upload dialog opens with an asset selection dropdown (no assets selected by default) and the same sheet preparation instructions. Click **Bulk Upload For Multiple Assets** to proceed, or **Cancel** to exit.

![Asset Mapping tab active, showing asset list and bulk upload option](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_9jXgETWxXeB22iCx6/eda446d9bc8659072c92707d9a50223basset_mapping_tab_active__showing_asset_list_and_bulk_upload_option.jpg)

#### Uploading fault code mappings for zones

Click the **Zone Mapping** sub-tab to switch to the zone list. Each zone (for example, Zone120, Zone125, Zone130, Zone140) shows its last updated date and options to **Download Existing Mapping** or **Upload New Mappings**. The **Upload Zone Fault Sheet** button is also available on this screen.

### Managing loss types

Navigate to **Settings > Loss Types** to view a table of all configured loss types. Each row displays the **Loss Type**, a **Loss Type Description**, an **On/Off** toggle, and a **Manage** (edit) option.

Example loss types include:

* Equipment Failure
* Fixture Failure
* Setup & Adj (w/load)
* Cutting Tool change
* Start up (JH)
* Minor Stoppage
* Reduced Speed
* Defect & Rework (Q prob)

You can scroll down to view additional entries. Use the **On/Off** toggle on any row to enable or disable that loss type, or click the **Edit** icon to modify its name or description.

![Loss Types screen with list of loss types, toggles, and edit options](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_vemHdLM5A5ot9T54J/e46bab459891a8683a592a31e2b0380closs_types_screen_with_list_of_loss_types__toggles__and_edit_options.jpg)

### Configuring process parameters

Navigate to **Settings > Process Parameters** to view a table of all configured parameters. Each row includes the following columns: **Parameter**, **Machine**, **Part Type**, **Min**, **Max**, **Target**, **Unit**, **Aggregator**, **Type**, and **Edit**.

An example entry looks like this:

* **Parameter:** X2-NR4\_ok\_B3
* **Machine:** OP200\_1\_INSTATION
* **Part Type:** VWR2KA 10B759 KC \[CATL 55 PP]
* **Min:** 1 | **Max:** 1 | **Target:** 5
* **Aggregator:** Min
* **Type:** Quality

![Process Parameters screen with parameter list and edit options](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_LMGlMstqN8AvC8Uv2/7677eada7a87f5b9296a8676a78b2acbprocess_parameters_screen_with_parameter_list_and_edit_options.jpg)

#### Editing a process parameter

Click the **Edit** icon on any row to open the **Edit Process Parameter** dialog. You can update the following fields:

* **Parameter**, **Machine**, **Part Type**
* **Min**, **Max**, **Target**
* **Aggregator** — select from the dropdown (for example, Min)
* **Type** — for example, Quality Parameter
* **Unit** — select from the dropdown if needed

Click **Save** to apply your changes, or **Cancel** to discard them and close the dialog.

![Edit Process Parameter dialog with all editable fields](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_rm1b4df8iqU6O8aM9/a5189b0e7e979cb96d2884e4f8b81945edit_process_parameter_dialog_with_all_editable_fields.jpg)

![Edit Process Parameter dialog with Cancel button highlighted](https://cdn.docsie.io/workspace_60BraQNlXDwUqHnAk/file_8r3hx8QL8zYbVPLmf/0d83569841945a8509921d3c048bc058edit_process_parameter_dialog_with_cancel_button_highlighted.jpg)

> **Tip:** Use the **Aggregator** dropdown to choose how parameter data is aggregated — for example, selecting Min will use the minimum value recorded for that parameter.
